# Chatty-v2
A real time chat app using graphql style api.

# Usage
Please use below script to run the app.
```
git clone git@github.com:jiabailie/Chatty-v2.git

cd server
npm install
node index.js

cd public
npm install
npm run start
```

Then you can open http://localhost:3000/ in browser to use it, and you can use http://localhost:50000/graphql to test the graphql APIs.
